# Control Validation Methods

## Approach: Trust but Verify

### Techniques
- Sampling (select controls randomly)
- Walkthroughs (step-by-step validation)
- Evidence inspection (logs, tickets, records)

## Example Controls
- Access control → verify actual user access list
- Logging → review real log entries
- Incident response → review past incidents

## Outcome
Ensures controls are operational, not theoretical.
